# WIT - Basic Version Control System

מערכת פשוטה לניהול גרסאות המבוססת על Python וספריית Click. הפרויקט מאפשר לעקוב אחרי שינויים בקבצים, לשמור "צילומי מצב" (Snapshots) ולחזור לגרסאות קודמות.

## פקודות זמינות

- `init`: אתחול מאגר חדש (יוצר תיקיית `.wit`).
- `add <path>`: הוספת קובץ או תיקייה לאזור ההמתנה (Staging).
- `commit -m "message"`: שמירת המצב הנוכחי כגרסה חדשה עם מזהה ייחודי.
- `status`: הצגת מצב הקבצים (מה חדש, מה שונה ומה מוכן לשמירה).
- `checkout <commit_id>`: חזרה בזמן לגרסה ספציפית לפי ה-ID שלה.

## התקנה
